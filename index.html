Projects

Enhanced Traffic Management using YOLOv7

This roject optimizes traffic flow at junctions by using real-time vehicle detection with a YOLOv7 model. Trained on thousands of images, it adjusts signal timings based on traffic density and prioritizes emergency vehicles. Developed by a team of four, this system ensures efficient traffic management and faster emergency response.

SQuiLL - Web Based Database Management

SQUILL is a web-based database modifier designed for users with no SQL knowledge. By entering database login details and selecting the appropriate driver, users can connect to databases like MySQL and Oracle 19c. Once connected, they can create, delete, insert, update, and alter tables through an intuitive interface, making database management accessible to everyone.

WeatherNow - Dynamic Weather Forecast WebApp

WeatherNow is a weather forecasting website that displays the upcoming week's weather for any city. As users type a city name, suggestions appear, and upon selection, WeatherNow retrieves today's weather data from the OpenWeatherMap API. The response is filtered and shown on the website. For weekly forecasts, coordinates from OpenWeatherMap are used with the OpenMeteo API. The website also changes its background based on the weather, enhancing the user experience.

SonoShift - Noise Cancellation in Audio

Noise interference is a persistent issue in video and audio applications, affecting recording, telecommunication, and speech recognition. Our project, SonoShift, aims to develop a noise cancellation system using artificial intelligence and machine learning (AIML). This system analyzes audio data to efficiently remove unwanted noise, providing clear and enhanced audio output.
